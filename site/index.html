# Unix socket daemon + MCP server — talk to it from any AI tool

$ deskbrid daemon --mcp-port 9001 # socket + TCP
← daemon ready · Sway backend loaded (auto-detected)

$ deskbrid health # verify dependencies in one shot

Every control surface, one socket

🪟

Window Control

List, focus, close, minimize, maximize, move, resize, activate-or-launch. Layout profiles. Full programmatic window management across any desktop environment.

⌨️

Input Injection + uinput

Type text, press keys, combo shortcuts, mouse drag. Keyboard layout management — list, get, switch, add, remove layouts. Absolute pointer via uinput for pixel-perfect clicks. ydotoold, XTest, and Mutter RemoteDesktop pipelines.

🔌

MCP Server

85-tool Model Context Protocol server on stdin/stdout or TCP. Hermes Agent, Claude Code, Codex, Cursor — any MCP client can control the desktop. Zero external MCP crate dependency.

🤖

Hermes Agent Plugin

First-class Hermes Agent integration. Drop the plugin in, enable it, and all 85 tools appear in every agent session. Bundled skill teaches agents how to use the desktop. Auto-starts the daemon. Direct Unix socket mode — no MCP overhead.

🌳

AT-SPI2 Accessibility Tree

Browse the accessibility tree, find elements by role/name, click them, read text. Full BFS snapshot builder with filtering. Perfect for UI automation.

📷

Screenshots + OCR + Diff

Full-screen, window, and region captures. Optical character recognition. Pixel diffing to detect visual changes. Works on GNOME, KDE, Hyprland, COSMIC.

📋

Clipboard + History

Read/write clipboard with format negotiation. Clipboard history with timestamps. Primary and clipboard selections on Wayland.

🌐

Browser CDP Control

List tabs, navigate, evaluate JavaScript, take screenshots, click elements — all via Chrome DevTools Protocol. Control Chromium/Chrome/Firefox from your agent.

📁

File Operations

Read, write, copy, move, delete, mkdir, list — full filesystem access through the socket. No separate tool needed.

🎵

MPRIS Media + Audio

Play, pause, next, previous, volume control. Per-app audio management via pactl. Bluetooth device pairing through bluetoothctl.

🔔

System Controls

Notifications, systemd unit management, polkit actions, network config via nmcli, terminal PTY sessions, sysfs backlight control, thermal zone monitoring, CPU frequency and governor management, daemon rate limiting with audit log.

📺

Workspaces + Monitors

List and switch workspaces, move windows. Monitor control with xrandr/cosmic-randr. Multi-monitor awareness with per-monitor workspace isolation.

🎨

Color Picker + App Catalog

Sample pixel colors anywhere on screen. App catalog with process detection, confinement checks, and idle detection via input device timestamps.

Get started in 30 seconds

# One-shot install — auto-detects desktop + installs deps recommended
$ bash <(curl -fsSL https://deskbrid.patchhive.dev/install.sh)
# Download binary directly
$ curl -LO https://github.com/coe0718/deskbrid/releases/download/v0.12.0/deskbrid-x86_64-unknown-linux-gnu.tar.gz && tar xzf deskbrid-x86_64-unknown-linux-gnu.tar.gz && chmod +x deskbrid
# Or build from source
$ git clone https://github.com/coe0718/deskbrid.git && cd deskbrid && cargo build --release
# Start the daemon
$ deskbrid daemon

HERMES AGENT — PLUGIN

Copy the plugin into ~/.hermes/plugins/deskbrid/, enable it in config.yaml, restart Hermes. All 85 tools appear. Or use MCP mode with a 5-line config entry.

cp -r hermes/plugin ~/.hermes/plugins/deskbrid
🔌
HERMES AGENT — MCP (zero plugin)

Add to ~/.hermes/config.yaml and restart. Hermes discovers every tool automatically. No plugin code needed.

mcp_servers:
  deskbrid:
    command: deskbrid
    args: [mcp]
